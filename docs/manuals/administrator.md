# Руководство администратора — VITBON ККМ

## Установка

1. Скачайте APK или установите через Google Play
2. При первом запуске: авторизуйтесь как администратор (PIN: `9999`)
3. Перейдите в **Настройки → Оборудование**

## Подключение ККТ

### MSPOS-K
1. Настройки → Оборудование → MSPOS-K
2. Введите IP-адрес устройства (по умолчанию: `192.168.1.100`)
3. Нажмите «Тест соединения»
4. При успехе: появится статус ФН

### Нева 01Ф
1. Настройки → Оборудование → Нева 01Ф
2. Выберите «Встроенный ФН»
3. Тест соединения → ОК

## Облачный товароучёт

1. Настройки → Синхронизация
2. Введите адрес сервера: `https://api.vitbon.ru`
3. Введите API-ключ (из личного кабинета)
4. Нажмите «Синхронизировать»

## Управление пользователями

1. Настройки → Пользователи
2. Добавить: ФИО, ПИН, Роль
3. Роли: **Администратор** (полный доступ), **Старший кассир**, **Кассир**

## Активация модулей

### ЕГАИС
1. Настройки → Модули → ЕГАИС
2. Введите адрес УТМ: `http://localhost:8080`
3. Загрузите RSA-сертификат
4. Активировать

### Честный ЗНАК
1. Настройки → Модули → Маркировка
2. Подключите ЛМ ЧЗ (локальный модуль)
3. Активировать

## Статусы системы

В правом верхнем углу — индикаторы:
- 🌐 Интернет
- ☁️ Облако
- 🧾 ОФД
- 🍺 ЕГАИС *(если активирован)*
- 🏷️ ЧЗ *(если активирован)*
- 🔒 Лицензия

## Обновления

Обновления загружаются автоматически. Для принудительного обновления:
Настройки → О системе → Проверить обновления
