# Руководство по маркировке и ЕГАИС — VITBON ККМ

## ЕГАИС

### Приёмка накладных

1. ЕГАИС → Приёмка накладных
2. Система загрузит накладные от поставщика автоматически
3. Проверьте товар по факту
4. Нажмите «Подтвердить приёмку»

### Акт вскрытия тары

При продаже алкоголя из кег:
1. Продажа → выберите товар
2. Система автоматически сформирует акт вскрытия
3. Подтвердите

### Проверка возраста (Цифровой ID Max)

1. ЕГАИС → Проверка возраста
2. Сканируйте QR-код с цифрового паспорта покупателя
3. Дождитесь результата:
   - ✅ **Подтверждён** → продажа разрешена
   - ❌ **Не подтверждён** → продажа заблокирована

## Честный ЗНАК (маркировка)

### Продажа маркированного товара

1. Откройте смену
2. Сканируйте DataMatrix-код
3. Система проверит код в системе «Честный ЗНАК»
4. При статусе ✅ «Можно продавать» — завершите продажу
5. Код автоматически списывается

### Возврат маркированного

1. Возврат → введите номер чека
2. Повторно отсканируйте DataMatrix
3. Система вернёт товар в оборот

### Статусы кодов

| Статус | Значение | Действие |
|--------|---------|----------|
| ✅ Можно продавать | Код в обороте, не выбыл | Продажа разрешена |
| ❌ Не в обороте | Код не зарегистрирован | Продажа запрещена |
| ❌ Уже выбыл | Товар уже продан | Продажа запрещена |
| ⚠️ Истёк срок | Срок годности истёк | Продажа запрещена |
