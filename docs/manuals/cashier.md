# Руководство кассира — VITBON ККМ

## Вход

1. Введите ПИН (4–6 цифр)
2. Нажмите «Войти»

## Продажа

1. Откройте смену (если закрыта): Смена → Открыть смену
2. Сканируйте штрихкод или введите название товара
3. При необходимости измените количество
4. Выберите способ оплаты: 💵 Наличные / 💳 Карта / 📱 QR
5. Нажмите **ПРОДАТЬ**
6. Дождитесь печати чека

## Возврат

1. Нажмите «Возврат»
2. Сканируйте QR-код с чековой ленты ИЛИ введите номер чека
3. Выберите позиции для возврата
4. Нажмите «Оформить возврат»

## Открытие / закрытие смены

- **Открыть:** Смена → Открыть смену
- **Закрыть:** Смена → Закрыть смену (Z-отчёт)
- **X-отчёт:** Смена → X-отчёт (без закрытия)

## Внесение / изъятие

Смена → Внесение / Изъятие
- Введите сумму
- При необходимости — комментарий
- Подтвердите

## Отчёты

Смена → Отчёты
- Выберите период: смена / день / неделя / месяц
- Просмотрите выручку, количество чеков, средний чек
